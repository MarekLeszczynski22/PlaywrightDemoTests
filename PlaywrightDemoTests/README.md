# Playwright Demo Test Suite

Example Playwright UI test framework using C# and NUnit.

This project serves as a sandbox for experimenting with Playwright features and testing patterns. It is continuously updated as I explore different automation techniques and best practices.

## Overwiew
This project showcases automated testing for the [SauceDemo](https://www.saucedemo.com) e-commerce application using:
- **Playwright** - Modern browser automation framework
- **C#/.NET** - Programming language and runtime
- **NUnit** - Testing framework
- **Page Object Model** - Design pattern for maintainable tests
